    if 直接判斷手有沒有放下來:
        儲存當前的history，拿去做分析
        分析1: 找最靠近肩膀的點，然後切斷 -> 如果是沒有在畫框內肩膀的點，就不算
        分析2: 判斷折返點 (如果大於90度，就代表折返點，直接取前段)

    if 停住一段時間 and 在畫框外:
        儲存當前的history，拿去做分析
        分析1: 找最靠近肩膀的點，然後切斷 -> 如果是沒有在畫框內肩膀的點，就不算
        分析2: 直接判斷整段的角度

    if 停住一段時間 and 在畫框內:

            if sum(|dx|) >> sum(|dy|) and dx 先正在負: 往右滑
            if sum(|dy|) >> sum(|dx|) and dy 先負在正: 往上滑
            if sum(|dy|) >> sum(|dx|) and dy 先正在負: 往下滑
